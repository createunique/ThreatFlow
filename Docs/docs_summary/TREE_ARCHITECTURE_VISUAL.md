```
computeTreePathAnalyzers(rootId, leafId, nodes, edges):

  allPaths = []
  visited = Set()

  dfs(currentId, path):
    IF currentId in visited:
      RETURN  // Prevent cycles
    
    node = find node by currentId
    IF not node:
      RETURN  // Invalid node
    
    newPath = [...path]
    
    IF node.type === 'analyzer':
      newPath.push(node.analyzer)  // Collect analyzer
    
    IF currentId === leafId:
      allPaths.push(newPath)  // Found target leaf
      RETURN
    
    visited.add(currentId)
    
    FOR EACH edge WHERE edge.source === currentId:
      dfs(edge.target, newPath)  // Explore children
    
    visited.delete(currentId)  // Backtrack
  
  dfs(rootId, [])  // Start from File root
  
  // Merge all paths (unique analyzers)
  analyzers = Set()
  FOR EACH path in allPaths:
    FOR EACH analyzer in path:
      analyzers.add(analyzer)
  
  RETURN Array.from(analyzers)
```
